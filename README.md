# Hospital Management System Web API

Bu proje, hasta ve doktor kayıtlarını yöneten, **.NET 8.0** üzerinde geliştirilmiş bir RESTful Web API projesidir.

## Özellikler

* **.NET 8.0 & RESTful Web API:** Yüksek performanslı ve ölçeklenebilir API mimarisi.
* **Kimlik Doğrulama (JWT & Azure SDK):** JSON Web Token (JWT) ve Azure SDK entegrasyonu ile güvenli, bulut tabanlı kimlik doğrulama ve yetkilendirme altyapısı.
* **Entity Framework Core:** İlişkisel veritabanı modellemesi.
* **Performans Optimizasyonu:** Kritik CRUD operasyonlarında doğrudan **Stored Procedure** ve **LINQ** sorguları kullanılarak veritabanı işlem yükü ve yanıt süreleri optimize edilmiştir.
* **Swagger Entegrasyonu:** Tüm API uç noktaları Swagger arayüzü ile standartlara uygun dökümante edilmiştir.
* **MVC İstemcisi:** `HttpClient` kullanılarak ASP.NET Core MVC arayüzü ile Web API arasında uçtan uca kesintisiz entegrasyon sağlanmıştır.

## Kullanılan Teknolojiler

* **Backend:** C#, .NET 8.0, ASP.NET Core Web API, ASP.NET Core MVC
* **Veritabanı & ORM:** MS SQL Server, Entity Framework Core, LINQ, Stored Procedures
* **Güvenlik:** JWT (JSON Web Token), Azure SDK
* **Araçlar:** Git, Swagger, HttpClient
